## **간접배출(전력) – Indirect Emission(Electricity)**

### **요약**

본 방법론은 기업이 전력 사용과 관련된 Scope 2 간접배출을 산정하고 보고하는 방법을 안내합니다. 전력 사용량 데이터를 기록하고 수집하는 절차를 설명합니다.

이 방법론은 기업이 전력을 소비하는 시설을 운영하며, 해당 전력 사용으로 인해 발생하는 배출이 전력 공급자 측에서 발생하는 경우에 적용됩니다. 전력 소비로부터 발생하는 간접배출을 산정하도록 설계되었습니다. 전력은 일반적으로 한국전력공사(KEPCO)를 통해 공급되지만, 기업은 조직 경계 내에서 재생에너지를 자체 생산할 수도 있습니다.

본 방법론은 위치기반(Location-based)과 시장기반(Market-based) 두 접근 방식을 모두 지원하여, 기업이 GHG 프로토콜 및 국제 기준에 따라 구매전력과 재생에너지 인증서(REC)를 반영하여 배출량을 산정할 수 있도록 합니다.

### **적용 범위 (Scope)**

- Scope 2 간접 배출: 외부에서 공급받은 전력
- 해당 배출량은 공급업체의 Scope 3, Category 1 또는 2에 포함될 수 있습니다. 다만, Scope 3 배출을 산정할 때에는 별도의 Scope 3 방법론을 사용해야 합니다.


### **사용자 입력 데이터**

- 적용 보고연도
- 전력 사용량 
  - KEPCO ON <https://online.kepco.co.kr/>
  - 한국전력 에너지마켓플레이스 <https://en-ter.co.kr/main.do>
- 재생에너지 인증서 유형 (MWh)
  - 간접 PPA / 직접 PPA / REC / 지분 참여

### **참고사항 – 재생에너지 전력의 이용**

재생에너지 설비 또는 폐열 회수 설비 등에서 생산된 전력을 구분하여 증빙할 수 있는 경우, 해당 에너지원에 적용되는 배출계수를 사용하여 관련 온실가스 직접배출량을 산정할 수 있습니다.

태양광, 풍력, 수력, 해양, 지열, 바이오에너지 등 신·재생에너지원을 통해 생산된 전력이 신재생에너지공급의무(RPS) 대상이 아닌 경우, 유효한 재생에너지 사용 인증서를 아래의 방식 중 하나로 확보한 조건에서, 재생에너지 배출계수(일반적으로 0 tCO₂eq/MWh)를 적용하여 별도로 산정할 수 있습니다.                                                                                      |

|           |                                                                                                      |
| :-------: | :--------------------------------------------------------------------------------------------------: |
| 간접 PPA  |                              전기판매사업자를 통한 전력구매계약의 체결                               |

| 지분 참여 |                지분 참여를 통한 전력 및 신·재생에너지 공급인증서(REC) 구매계약의 체결                |

## Use case: 일반 기업의 전력 간접배출 산정

**시나리오 개요**

C사는 본사 사옥과 세 개의 제조 공장을 운영하고 있으며, 모든 시설은 한국전력공사(KEPCO)로부터 공급받은 전력을 사용한다. ESG 공시와 내부 에너지 효율 관리 목적을 위해, 회사는 매년 Scope 2 간접배출량을 산정한다.

**1) 데이터 수집**

우선순위 1 — 직접 계측

- 전력량계 등 법정계량기를 통해 각 사업장의 연간 사용량(MWh) 확보

우선순위 2 — 공급사 조회

- 전력량계 고장 또는 데이터 누락 시, 한전ON·에너지마켓플레이스 등을 통해 공급사 청구 내역 조회

**2) 재생에너지 사용분 확인**

- 사업장 내 태양광, 풍력 등에서 생산한 전력과 구매 전력 구분
- REC 또는 PPA 계약 등으로 재생에너지 사용이 검증된 전력의 경우, 시장기반 접근방식에서 회피배출을 반영하기 위해 해당 재생에너지에 적용되는 배출계수를 사용하여 배출량 산정

**3) 배출계수 적용**

- 위치기반 접근방식의 경우, 국가 전력 배출계수 (예: 0.4781 tCO₂eq/MWh) 사용
- 시장기반 접근방식의 경우, 검증된 재생에너지 전력과 나머지 전력 사용량을 구분 후, 검증된 재생에너지 전력 구매분에는 재생에너지 배출계수를 적용. 잔여 전력 사용량에 대해 잔여전력(residual mix) 배출계수가 제공되는 경우 해당 값을 적용. 잔여전력 배출계수가 없는 경우, 위치기반 접근방식에서 사용하는 국가 계통 배출계수를 동일하게 적용.

**4) 배출량 산정 절차**

- 총 전력 사용량(MWh)을 산정하고, 검증된 재생에너지 인증서 구매량을 구분한다.
- 위치기반 접근방식 산정식을 적용하여 총 위치기반 배출량을 계산 = Q × EFj

  - Q : 전력 사용량(MWh)
  - EFj : 전력 배출계수(tGHG/MWh)
- 시장기반 접근방식 산정식을 적용하여 총 시장기반 배출량을 계산 = ∑(Q<sub>R</sub> × EF<sub>R</sub>) + (Q<sub>N</sub> × EF<sub>j</sub>)

산정 예시:

- 전력 소비량: 15,000 MWh

- 재생에너지 인증서 구매량:

  - 간접 PPA: 300 MWh

  - 직접 PPA: 0 MWh

  - REC 구매: 1,200 MWh

  - 지분 참여: 0 MWh

- EF<sub>j</sub>: 0.4781 tCO₂eq/MWh

- EF<sub>R</sub>: 0 tCO₂eq/MWh

- 위치기반 배출량: 15,000 MWh × 0.4781 tCO₂eq/MWh = 7,171.5
  tCO₂eq

- 시장기반 배출량: (300 MWh x 0 tCO₂eq/MWh) + (1,200 MWh x 0
  tCO₂eq/MWh) + \[(15,000 MWh - 300 MWh - 1,200 MWh) x 0.4781
  tCO₂eq/MWh\] = 6,454.35 tCO₂eq

**5) 보고 및 활용**

- ESG 보고서 Scope 2에 반영
- 사업장별 전력 효율 개선 KPI 설정
- 재생에너지 사용 비율 관리 및 확대 전략 수립
