# Previews the policy from IPFS without loading it into the local DB.

```yaml
POST /external-policies/preview
```

**Headers**

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |
