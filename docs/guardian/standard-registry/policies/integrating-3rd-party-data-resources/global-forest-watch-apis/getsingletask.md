# getSingleTask

GET /task/:task\_id

Get single tasks by task ID

**Headers**

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

**Body**

| Name     | Type   | Description |
| -------- | ------ | ----------- |
| task\_id | string | Task ID     |
