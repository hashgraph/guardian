# getSingleTask

<mark style="color:green;">`GET`</mark> `/task/{task_id}`

Get single tasks by task ID

**Headers**

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

**Body**

| Name     | Type   | Description |
| -------- | ------ | ----------- |
| task\_id | string | Task ID     |
