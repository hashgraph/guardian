# Creating new Module

## Creates a new module.

<mark style="color:green;">`POST`</mark> `/modules`

#### Request Body

| Name                               | Type | Description                               |
| ---------------------------------- | ---- | ----------------------------------------- |
| <mark style="color:red;">\*</mark> | Json | Object that contains module configuration |
